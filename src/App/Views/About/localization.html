Get started with Localization for Metamorphosis

Localization provides an interface to translate languages to content readers native language.

Using Localization

Localization is added to the global scope of twig, and can be accessed using the {{ translation.KEY_WORD }}

In the following example we are getting the header title

            {{ translation.fields_header_title }}

All the translation files are located in ./Core/Localization/Languages/*

To access any key in the translation array, just make sure it is present in the language files, and use the syntax above in an html template

Creating Localization

To create a new language which can be translated add a new language file in ./Code/Localization/Languages/shortname.php where shortname is the HTTP_ACCEPT_LANGUAGE in two-three letters.

Example below uses the English translation class

            namespace Core\Localization\Languages;

            use Core\Localization\Localization;
            use Core\Localization\iLanguage;

            /**
             * English localization class
             *
             * @version: PHP: 8.1
             * @en
             */
            class en extends Localization implements iLanguage {
                public static array $Language = array(
                    /**
                     * The naming convention should be based on the following
                     *
                     * [COMPONENT_LOCATION_NAME]
                     */
                    "fields_header_title" => "Hello World!"
                );

                /**
                 * Static get the language array
                 *
                 * @return array
                 */
                public static function getLanguage(): array {
                    return static::$Language;
                }
            }
